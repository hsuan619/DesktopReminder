# 📝 DesktopReminder (側邊欄待辦小幫手)

## 忘記當天要處理的事嗎？
**DesktopReminder** 是一個在您的瀏覽器上漂浮的專屬待辦清單小幫手。
無論您正在瀏覽哪個網頁，它都會像一顆便利的「小白點」陪伴在側。隨時點擊，隨時記錄，讓每個靈感與待辦事項都不再溜走！

---

## ✨ 主要功能 (Features)

*   🗓️ **七天日程規劃**：可以記錄時間，除了當日任務，最長可建立並管理未來 **7 天內** 的待辦事項。
*   ⏰ **智慧紅點預警**：為任務設定預定時間後，在行程即將到來的 **前 10 分鐘**，小白點會自動跳出醒目的「🔴 紅點動畫」提醒您！
*   🖱️ **全域懸浮與自由拖曳**：注入在所有網頁之上，您可以自由將小白點拖曳到畫面的任何角落，絕不干擾閱讀。
*   ⌨️ **一鍵快捷呼叫**：支援全域快捷鍵，按下 `Ctrl + Shift + A` (Mac 為 `Command + Shift + A`) 即可瞬間喚醒面板縮放。
*   ⚠️ **逾期追蹤機制**：過去忘記打勾的任務，會自動亮起紅框並顯示「延遲」警告，跨日自動結轉到今日清單，直到您完成它為止。

## 🚀 快速安裝與使用 (Installation)

1.  **取得檔案**：將此專案下載並解壓縮到您的電腦中。
2.  **開啟擴充功能管理**：在 Chrome 瀏覽器網址列輸入 `chrome://extensions/` 並按下 Enter。
3.  **開發人員模式**：在頁面的右上角，將「**開發人員模式**」的開關打開。
4.  **載入專案**：點擊左上角的「**載入未封裝項目**」(Load unpacked) 按鈕。
5.  **選擇資料夾**：選擇剛剛解壓縮後的專案資料夾 (包含 `manifest.json` 的那一層)。
6.  **享受體驗**：安裝完成！請開啟一個全新的網頁 (或重新整理現有網頁)，您就會在右下角看到專屬的待辦小白點了！

## 🔒 隱私與資料安全 (Data Privacy)

**我們極度重視您的隱私，且保證絕對安全。**

DesktopReminder 是一個 **100% 離線運作 (Offline-First)** 的工具：
*   ✅ 所有的任務、時間與行程紀錄，都**僅存儲在您個人連線裝置的瀏覽器本機內 (Chrome Local Storage)**。
*   ❌ 絕對沒有連接任何外部雲端伺服器或資料庫。
*   ❌ 不會收集、分析或上傳您的任何瀏覽紀錄或任務數據。

您可以毫無顧忌地將公司機密排程、重要的個人私事或是各種天馬行空的靈感記錄在裡面！

## 💡 疑難排解與提示

*   **遇到 `Extension context invalidated` 錯誤？**
    如果我們發布了新版本且您在背景更新了它，原本開啟的舊網頁可能會失去連線。系統會友善地跳出提示，此時只要 **「重新整理 (F5)」** 該網頁即可恢復生機。
*   **如何修改快捷鍵？**
    如果您覺得 `Ctrl + Shift + A` 衝突了，可以前往 `chrome://extensions/shortcuts` 頁面，找到 DesktopReminder 來自定義您最順手的組合鍵。
